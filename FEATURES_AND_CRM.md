# Features & CRM integrations

## Core features

- **Inbound call answering, 24/7** — no hold music, no voicemail; the agent
  picks up and handles the call in real time.
- **Booking** a new appointment directly into the clinic's calendar/CRM.
- **Rescheduling** an existing appointment.
- **Cancelling** an appointment.
- **Caller identification** — recognizes returning callers and looks up
  their existing appointments instead of asking them to repeat everything.
- **Multi-language** — the agent speaks the clinic's configured language
  (currently French, English, Russian, Arabic).
- **Call history & transcripts** for clinic staff, reviewable from the
  dashboard.
- **Analytics** on call volume, outcomes and missed-call recovery.
- **Role-based dashboard access** for clinic owners, managers, front-desk
  staff and accountants.
- **Multi-currency billing** — clinics are billed in their local currency
  (EUR, GBP, USD, RUB).
